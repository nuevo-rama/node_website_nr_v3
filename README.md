# node_website_nr_v3
Node Website

Development guide

1. npm init
2. npm i express
3. npm i ejs
4. npm i morgan
5. npm i nodemon -D (-D dependency option)

5. Create 'src' folder
6. Create the entry point 'index.js'

7. In package.json

"scripts": {
    "start": "node src"
    "dev": "node src" 
  }

8. For run in dev mode: 

'npm run dev'

9. Create .gitignore

'node_modules'
